# Demo · 双轴设计模式

**本 module 没有可跑 demo。**

双轴框架的本质是「设计语言 + 选型方法」，运行时演示反而稀释概念。这个 module 教三件事：

1. 看懂自家 agent 系统在矩阵上的位置
2. 读得懂别人 agent 源码（逆向五步法）
3. 知道下一步该补哪个模式（盲区识别）

这三件事的「演示」靠 PPT + 讲解 + 学员对自己系统的现场标注完成——不靠跑代码。

## 真正能演示的「现场练习」

如果工作坊有时间做练习，做下面这个：

1. 让学员打开自家任意一个 agent 项目源码
2. 30 秒内回答：「主循环在哪行」（**Detect**）
3. 60 秒内回答：「这是 6 种拓扑里哪种」（**Classify**）
4. 用矩阵 PNG 在自己系统上标出走了哪几个格子（**Map**）
5. 看哪几列、哪几行格子全空 —— 那就是当前系统的盲区

这个练习不需要任何 demo 代码。一张矩阵图 + 学员自己的项目，10 分钟出结果。
